# 🖼️ Image Caption Generator

This project generates captions for images using Deep Learning (CNN + LSTM).

## 🚀 Features
- Extracts image features using a pretrained CNN (ResNet)
- Generates captions using LSTM
- Trained on image-caption dataset
- Implemented using PyTorch

## 🧠 Model Architecture
- Encoder: CNN (ResNet)
- Decoder: LSTM (RNN)

## 📊 Example
Input: Image of a dog on a beach  
Output: "A dog running on the beach"

## 🛠️ Tech Stack
- Python
- PyTorch
- NumPy
- PIL
